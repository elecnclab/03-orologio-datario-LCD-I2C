# 03-orologio-datario-LCD-I2C

Progetto Arduino: orologio con datario basato su RTC DS3231, display LCD I2C e encoder.

## Struttura

- `firmware/03-orologio-datario-LCD-I2C/03-orologio-datario-LCD-I2C.ino`
- `docs/schemi/`
- `docs/note-tecniche/`
- `media/foto/`
- `media/github-images/`
- `hardware/stl/`

## Avvio rapido

1. Apri lo sketch `.ino` nella cartella `firmware/03-orologio-datario-LCD-I2C`.
2. Verifica le librerie richieste in Arduino IDE.
3. Compila e carica su Arduino.
